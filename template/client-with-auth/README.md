# Client With Auth Template

A Next.js starter template with shadcn/ui authentication pages and a minimal sidebar dashboard.

## Features

- Login & Register pages (shadcn/ui forms)
- Dashboard with sidebar navigation (shadcn/ui sidebar)
- Placeholder pages: Analytics, Users, Content, Settings
- Tailwind CSS v4 with shadcn/ui design tokens
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you'll be redirected to the login page.

## Adding Auth

This template provides the UI only. Wire in your preferred auth solution:

- [NextAuth.js](https://next-auth.js.org/)
- [Clerk](https://clerk.com/)
- [Auth.js](https://authjs.dev/)
- Custom JWT / session-based auth
