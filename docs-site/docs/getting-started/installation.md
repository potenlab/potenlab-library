# Installation

Install `@potenlab/ui` and its peer dependencies.

## npm

```bash
npm install @potenlab/ui
```

## bun

```bash
bun add @potenlab/ui
```

## Peer Dependencies

The library requires the following peer dependencies:

| Package | Version |
|---------|---------|
| `react` | `>= 18` |
| `react-dom` | `>= 18` |
| `tailwindcss` | `>= 4` |

### Optional Peer Dependencies

| Package | Required for |
|---------|-------------|
| `@tanstack/react-table` | `DataTable` component |
| `next-themes` | Dark mode with Next.js |

```bash
npm install @tanstack/react-table next-themes
```
