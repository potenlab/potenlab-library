# Badge

An inline label component with multiple visual variants. Supports `asChild` for polymorphic rendering.

## Demo

<StorybookEmbed storyId="ui-badge--default" height={100} />

## Import

```tsx
import { Badge } from "@potenlab/ui";
// or
import { Badge } from "@potenlab/ui/components/ui";
```

## Usage

```tsx
<Badge variant="default">Active</Badge>
<Badge variant="green">Completed</Badge>
<Badge variant="destructive">Error</Badge>
```

## Props

| Prop | Type | Default |
|------|------|---------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link" \| "green"` | `"default"` |
| `asChild` | `boolean` | `false` |

Extends all native `<span>` HTML attributes.
